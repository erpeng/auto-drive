## 主题

<div class="index-grid">
<a class="index-card" href="/auto-drive/themes/l2-l4/">
  <span class="index-card-title">L2 与 L4</span>
  <span class="index-card-desc">当前语料最尖锐的观点之一，仍然是 L2 和 L4 不是天然一条直线上的前后阶段 。但新的两篇材料让分歧更复杂了: 现在已经不只是“断裂派”对“桥接派”，还出现了更明确的“连续派”。</span>
</a>
<a class="index-card" href="/auto-drive/themes/robotaxi/">
  <span class="index-card-title">Robotaxi</span>
  <span class="index-card-desc">Robotaxi 在当前语料里不是“自动驾驶的一个应用”，而是检验 L4 是否成立的最终商业场景。它既是技术问题，也是单位经济模型问题。</span>
</a>
<a class="index-card" href="/auto-drive/themes/note-0ef0d64de9/">
  <span class="index-card-title">技术路线</span>
  <span class="index-card-desc">当前语料里真正重要的技术路线之争，已经不只是“纯视觉还是激光雷达”，也不只是“端到端是不是先进”，而是 系统究竟要不要继续保留中间翻译层，以及世界模型到底应该长在车上还是长在云端 。</span>
</a>
<a class="index-card" href="/auto-drive/themes/note-30fb7c170c/">
  <span class="index-card-title">智驾芯片与生态</span>
  <span class="index-card-desc">如果说 Robotaxi 和城市 NOA 是舞台上的玩家，那么智驾芯片就是决定舞台能搭多大的地基。新的两篇材料把这件事又往前推了一步： 芯片不只是算力表，更是工程体系、生态选择和商业位置的放大器 。</span>
</a>
<a class="index-card" href="/auto-drive/themes/note-ab41bf10ba/">
  <span class="index-card-title">激光雷达</span>
  <span class="index-card-desc">在当前语料中，激光雷达不是一个孤立硬件品类，而是自动驾驶“确定性”问题的一个答案。</span>
</a>
</div>

## 公司

<div class="index-grid">
<a class="index-card" href="/auto-drive/companies/momenta/">
  <span class="index-card-title">Momenta</span>
  <span class="index-card-desc">Momenta 在当前语料里最像一台 把自动驾驶量产化的操作系统 。它不只是在做模型，而是在努力把车企、车型、数据、组织和量产节奏压成一条可复制主线。鲜活之处在于，它几乎把所有复杂性都翻译成“体系产品化”问题。</span>
</a>
<a class="index-card" href="/auto-drive/companies/waymo/">
  <span class="index-card-title">Waymo</span>
  <span class="index-card-desc">Waymo 在这套语料里几乎不是一家公司，而是 一把尺子 。很多文章并不直接分析它，却默认用它来校准什么叫真正的 L4、什么叫无人运营、什么叫已经坐上牌桌。它的鲜活之处不在品牌叙事，而在于它反复被别人拿来当“不得不承认的现实”。</span>
</a>
<a class="index-card" href="/auto-drive/companies/note-c19cc3b27f/">
  <span class="index-card-title">卓驭</span>
  <span class="index-card-desc">卓驭现在最鲜活的地方，不是“前大疆项目分拆”这个标签，而是它像一个 离开工程温室后，被迫重学商业与客户化生存的学霸型公司 。沈劭劼的访谈把它的张力写得很清楚: 工程底子很强，商业肌肉几乎空白；过去靠规则和机器人学派维持确定性，后来又不得不承认打不过纯数据驱动，选择删库重练。</span>
</a>
<a class="index-card" href="/auto-drive/companies/note-217712ba17/">
  <span class="index-card-title">地平线</span>
  <span class="index-card-desc">地平线表面卖芯片，骨子里想拿的是 产业里的默认接口权 。它不是满足于做“国产替代元件”的公司，而是一直想从芯片走向工具链、算法、生态，再走向一种类似安卓 / Wintel 的秩序位置。它最鲜活的地方，就是野心和现实之间始终存在的那条张力线。</span>
</a>
<a class="index-card" href="/auto-drive/companies/note-fd360bc0cd/">
  <span class="index-card-title">小马智行</span>
  <span class="index-card-desc">小马智行最鲜活的地方，是它不像一间“顺着市场热点调整姿势”的公司，而像一间 死守单一目标函数的 L4 专门学校 。它宁愿显得慢，也不愿把问题改写成更容易被市场理解的样子。楼天城和彭军几乎把整家公司都写成了一种方法论姿态: 不被 L2 稀释，不被端到端热词诱惑，不把 demo 当进展。</span>
</a>
<a class="index-card" href="/auto-drive/companies/note-c86bb992e0/">
  <span class="index-card-title">小鹏汽车</span>
  <span class="index-card-desc">小鹏汽车在这轮新材料里最值得单独读的地方，不是“车企也在做智驾”，而是它越来越像一间 把卖车业务当成物理 AI 训练场和资本引擎来使用的自研公司 。刘先明这篇访谈让小鹏的气质变得更清楚了: 它并不满足于做一个高阶辅助驾驶体验不错的车厂，而是在赌“极简模型 + 大数据 + 大算力 + 长时间耐心”能不能把车企自研推成一条通向更通用智能的主线。</span>
</a>
<a class="index-card" href="/auto-drive/companies/note-eac959b5c9/">
  <span class="index-card-title">文远知行</span>
  <span class="index-card-desc">文远知行在这批新语料里明显变立体了。它不再只是一个“我也在牌桌上”的声明样本，而更像一家 试图同时证明两件事的双线公司 : 一方面要证明自己是全球少数真正把 L4 跑起来的玩家，另一方面又要证明自己能把 L4 积累翻译成 L2++ 量产能力。韩旭的两篇访谈把这家公司最鲜活的气质写出来了: 它讨厌“终局论”，更相信技术范式会继续变，组织必须同时把 infras</span>
</a>
<a class="index-card" href="/auto-drive/companies/note-3f426f96e0/">
  <span class="index-card-title">特斯拉</span>
  <span class="index-card-desc">特斯拉在这套语料里最像一种 叙事压缩机 : 它试图把“汽车制造、FSD、芯片、自研模型、Robotaxi、AI 估值”压到同一个资产上。于是它总同时存在于三张桌上: 产品桌、技术桌、资本市场桌。鲜活之处正在这里: 它既可能是未来默认形态，也可能是被未来提前计价的样板。</span>
</a>
<a class="index-card" href="/auto-drive/companies/apollo-go/">
  <span class="index-card-title">百度 Apollo Go</span>
  <span class="index-card-desc">百度 Apollo Go 在当前语料里更像一台 公众感知机器 。它未必是被描述最细的公司，却是把 Robotaxi 从行业内行话推到普通人视野的那一位。武汉萝卜快跑出圈的意义，不只是订单和车队，而是让“无人化”第一次以社会事件而非技术新闻的形式出现。</span>
</a>
<a class="index-card" href="/auto-drive/companies/note-035bc23f3b/">
  <span class="index-card-title">禾赛</span>
  <span class="index-card-desc">禾赛最像一台 把激光雷达从“贵而脆的实验室器件”压成工业部件的制造机器 。它的鲜活之处不在情怀，而在一种很中国式、但又很工程化的信念: 真正的降本不是嘴上说规模，而是把设计、芯片化和自动化生产一起做穿。</span>
</a>
<a class="index-card" href="/auto-drive/companies/note-63ddce497a/">
  <span class="index-card-title">轻舟智航</span>
  <span class="index-card-desc">轻舟智航最值得单独读的地方，是它不像行业里最吵的玩家，却越来越像一间 把有限算力、有限人手和国产芯片生态榨到极致的识别型方案商 。于骞这篇访谈把它的公司气质讲得很透: 它不靠堆最大团队，不靠堆最贵平台，也不靠酒桌关系，而是靠更早识别芯片边界、把工程做穿、再用结果去换客户信任。</span>
</a>
</div>

## 人物

<div class="index-grid">
<a class="index-card" href="/auto-drive/people/note-28f6fd5edb/">
  <span class="index-card-title">于骞</span>
  <span class="index-card-desc">于骞最鲜活的地方，是一种很少见的组合: Waymo 出身的技术理想主义 + 做 2B 供应商之后被现实磨出来的成本纪律 。他既会认真谈“世界模型 + 强化学习”，也会反复提醒自己和团队，对主机厂来说没有什么比卖车更重要。读他的访谈，能感觉到这不是一句讨好客户的话，而是他真把这条苦生意内化成了方法论。</span>
</a>
<a class="index-card" href="/auto-drive/people/note-dcfd2ee796/">
  <span class="index-card-title">余凯</span>
  <span class="index-card-desc">余凯最鲜活的地方，不是“科学家创业”，而是 一个极强社会性的科学家 。他既享受站在边缘押注新范式，也明显享受在复杂关系网里升级打怪。你读他的访谈，会感觉他并不羞于谈江湖、谈认人、谈破局，甚至会把这些能力看成和技术判断同等重要的生存器官。</span>
</a>
<a class="index-card" href="/auto-drive/people/note-1e8ce634d4/">
  <span class="index-card-title">刘先明</span>
  <span class="index-card-desc">刘先明最鲜活的地方，不是“小鹏花钱最多的人”这个标签，而是他像一个 把自动驾驶重新定义成物理 AI 缩放问题的架构派负责人 。他在 Cruise 看过重规则、重激光雷达、重运维修补的旧范式，也亲眼被 ChatGPT 的简单架构 + 大参数量 + 大数据量震过一下，所以他现在最强的气味不是“把车开稳”，而是“把中间多余的东西都拆掉，让模型直接从物理世界里学”。</span>
</a>
<a class="index-card" href="/auto-drive/people/note-324f75c0cf/">
  <span class="index-card-title">彭军</span>
  <span class="index-card-desc">彭军给人的感觉，不像那种靠个人 charisma 往前冲的 CEO，更像一位 对复杂系统有强耐受度的总工程师型创业者 。他讲话时最鲜明的特点，是不断把外界热词拆回到问题本身，然后提醒你: 真正难的从来不是某个单点技术，而是把技术、车辆、政府、用户、运营和资本市场一起接成一条能活下去的链。</span>
</a>
<a class="index-card" href="/auto-drive/people/note-576d343d7f/">
  <span class="index-card-title">曹旭东</span>
  <span class="index-card-desc">曹旭东给人的第一感觉，不是锋利，而是 把复杂系统产品化的人 。他不像楼天城那样先切问题边界，也不像余凯那样先谈生态和江湖；他更像在不停问一件事: 这么复杂的组织、客户、车型、数据和量产，能不能被压成一条主线、一个体系、一个飞轮。</span>
</a>
<a class="index-card" href="/auto-drive/people/note-58b3bc42d2/">
  <span class="index-card-title">李一帆</span>
  <span class="index-card-desc">李一帆不像那种“血战出来”的创业者，更像一个 高顺滑度的工程叙事者 。他讲话里很少有苦大仇深，更多是把复杂技术翻译成朴素图像，再把行业机会放进国家、制造和设计降本的长期结构里。鲜活点在于: 他看起来温和，但对“什么才是真问题”有很清晰的筛选。</span>
</a>
<a class="index-card" href="/auto-drive/people/note-3307c75e01/">
  <span class="index-card-title">楼天城</span>
  <span class="index-card-desc">楼天城像一种很少见的技术创始人: 他不是先适应市场语言，再解释自己；而是先把问题定义得极窄极硬，然后要求世界按那个定义来接受检验。读他的访谈，最强烈的感觉不是“善辩”，而是 对目标函数的洁癖 。在他这里，L4 必须按 L4 的方式解，别的答案再热闹，也只是别的问题。</span>
</a>
<a class="index-card" href="/auto-drive/people/note-9c31fdd874/">
  <span class="index-card-title">沈劭劼</span>
  <span class="index-card-desc">沈劭劼像一种很少见的创始人原型: 学术强者被市场强行社会化后的二次进化体 。他的人物鲜活感不在于“聪明”，而在于那种非常具体的失重感: 从大疆工程温室搬出来，开始面对钱不够、客户不等人、规则写到绝望、团队要不要断后路这些问题。读他的访谈，能明显感觉到一个机器人学派工程师被现实逼着承认旧方法不够了。</span>
</a>
<a class="index-card" href="/auto-drive/people/note-48a288cf6a/">
  <span class="index-card-title">王皓俊</span>
  <span class="index-card-desc">王皓俊给人的感觉，是一种 把技术公司翻译给经营世界听的人 。他不像楼天城和彭军那样不断重申技术边界，而是把小马智行已经做出来的东西，翻译成 UE、Top line、扩车、城市复制、毛利结构和监管定价这些可以被 CFO 理解、也必须由 CFO 扛住的语言。鲜活点在于: 他不热衷于“宣布胜利”，更像是在给市场校准预期。</span>
</a>
<a class="index-card" href="/auto-drive/people/note-4da7fe5a50/">
  <span class="index-card-title">韩旭</span>
  <span class="index-card-desc">韩旭给人的第一感觉，不是那种会把路线讲成宿命的人，反而像一个 对“技术还有下一轮变化”始终保留余地的基础设施派 CEO 。他讲话的鲜活处在于，明明身处一个很爱讲“终局”“决赛圈”“只剩三家”的行业，他却不断把话题拉回到数据生产率、组织训练机制和服务化纪律上。</span>
</a>
</div>
