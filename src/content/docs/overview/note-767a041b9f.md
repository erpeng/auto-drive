## 桌子一：L4 / Robotaxi 牌桌

这一桌的入场门槛不是 PPT、demo 或功能体验，而是三个更硬的条件：

- 真正无人化
- 面向公众开放运营
- 有一定车队规模和持续运营能力

在这张桌上，当前语料里至少存在两种说法：

- [楼天城](/auto-drive/people/note-3307c75e01/) 的说法更严格：全球真正坐在桌上的主要是 [Waymo](/auto-drive/companies/waymo/)、[百度 Apollo Go](/auto-drive/companies/apollo-go/)、[小马智行](/auto-drive/companies/note-fd360bc0cd/)，而 [特斯拉](/auto-drive/companies/note-3f426f96e0/) 即便是 serious player，也还没完全坐上桌。
- [文远知行](/auto-drive/companies/note-eac959b5c9/) 的韩旭则给出另一套框架：国内真正能纯无人运营的只有三家，而他显然把文远也算在内。

这不是简单的谁对谁错，而是口径不同。一个更强调全球对标和车队成熟度，一个更强调中国本土无人运营资格。

## 桌子二：L2 / 城市 NOA 量产牌桌

这一桌比拼的是：

- 上车车型数
- 客户数量
- 城市 NOA 体验
- 成本与交付效率
- 对车企的组织适配能力

当前语料的主流判断是：

- 第一梯队是华为乾崑与 [Momenta](/auto-drive/companies/momenta/)
- [地平线](/auto-drive/companies/note-217712ba17/) 正试图从芯片优势延伸到整套系统交付
- [卓驭](/auto-drive/companies/note-c19cc3b27f/) 是刚从“大疆温室”里完全走出来的新变量，技术观与 Tesla 式端到端更新更接近，但商业肌肉和客户化能力仍在补课
- 车企自研是持续存在的威胁和变量

## 为什么牌桌会错位

- L4 玩家容易把 L2 看成“另一项运动”。
- L2 玩家则更容易用量产节奏、车型数和客户数来定义竞争胜负。
- 投资市场又经常把两者都映射为“自动驾驶”一个词。

因此，很多争论本质上是在争“你在说哪张桌子”。

## 当前值得记录的分歧

- 特斯拉算不算 L4 牌桌玩家：楼天城偏谨慎，资本市场和部分评论者更积极。
- 文远知行是否应被纳入中国 L4 第一梯队：韩旭的自述明确认为应当纳入。
- L2 与 L4 是否共享同一条进化曲线：L4 从业者普遍否认，Momenta 试图部分打通。
- 城市 NOA 新变量是否来自老牌供应商二次转身：沈劭劼的访谈说明，卓驭正试图用“删库重练”的方式重新拿到下一轮入场券。([对话卓驭沈劭劼：从大疆到百亿智驾公司 CEO 的十年之路](/auto-drive/raw/ceo/))
