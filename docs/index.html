自动驾驶 Wiki Index

这是当前 Obsidian 自动驾驶知识库的入口页。当前语料以 2022-2026 年的中文行业访谈、公司分析和赛道评论为主，重心集中在 自动驾驶行业总览、Robotaxi、L2 与 L4、技术路线、智驾芯片与生态、激光雷达。

总览

  • 自动驾驶行业总览: 当前语料所呈现的行业地图，包括 Robotaxi、L2 量产、芯片和传感器三条主线。
  • 自动驾驶牌桌: 区分 L4/Robotaxi 牌桌与 L2/城市 NOA 牌桌，并记录不同受访者的分歧。

主题

  • Robotaxi: Robotaxi 的经济账、本地扩张逻辑、无人化门槛与终局想象。
  • L2 与 L4: 为什么多位 L4 从业者认为 L2 与 L4 不是线性升级关系。
  • 技术路线: 端到端、模仿学习、世界模型、强化学习、数据飞轮之间的关系。
  • 智驾芯片与生态: 地平线、英伟达、车企自研与“安卓式生态”叙事。
  • 激光雷达: 激光雷达作为“主动的眼睛”的价值、降本路径与中美分化。

公司

  • Waymo: 全球 L4 标杆，很多讨论都把它当作事实上的技术和运营基准。
  • 特斯拉: FSD 与 Robotaxi 的叙事中心，但是否算真正的 L4 玩家存在争议。
  • 百度 Apollo Go: 武汉无人化出圈，构成中国 Robotaxi 拐点认知的重要来源。
  • 小马智行: L4 坚定派，围绕楼天城与彭军的访谈构成一条清晰技术主线。
  • Momenta: 中国城市 NOA 第一梯队代表，以数据飞轮和“两条腿”著称。
  • 地平线: 从智驾芯片走向算法交付，核心命题是生态能否升级为系统能力。
  • 卓驭: 大疆体系分拆出的新变量，技术底子强，正在补商业肌肉和客户化能力。
  • 文远知行: 以韩旭视角补充“国内真正 L4 玩家”名单与海外扩张逻辑。
  • 禾赛: 激光雷达基础设施公司，关键能力在制造、ASIC 化与成本曲线。

人物

  • 楼天城: L4 世界观最强烈的代表人物之一，强调无人化、世界模型与“牌桌”标准。
  • 彭军: 小马智行 CEO，兼具工程纪律与经营复杂度意识，是公司长期主线的总协调者。
  • 王皓俊: 小马智行 CFO，把技术主线翻译成 UE、Top line、城市复制和经营节奏的语言。
  • 曹旭东: Momenta 的组织和产品方法论核心，强调数据驱动与客户共创。
  • 韩旭: 文远知行 CEO，反终局论、重 infrastructure，相信路线仍在持续流动。
  • 沈劭劼: 卓驭 CEO，从机器人学派转向数据驱动，鲜明地经历了从工程温室到市场旷野的转换。
  • 余凯: 地平线的生态、创业与组织观入口。
  • 李一帆: 禾赛的产品化与制造观入口，解释激光雷达如何从贵而脆走向规模化。
