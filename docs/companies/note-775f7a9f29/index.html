地平线

地平线表面卖芯片，骨子里想拿的是产业里的默认接口权。它不是满足于做“国产替代元件”的公司，而是一直想从芯片走向工具链、算法、生态，再走向一种类似安卓 / Wintel 的秩序位置。它最鲜活的地方，就是野心和现实之间始终存在的那条张力线。(地平线还是想当安卓, 地平线余凯：技术从来没有成为过护城河)

事实底稿

  • 在基础辅助驾驶芯片市场，地平线凭借性价比、本土供应链和开放合作切走了 Mobileye 的大量位置。(地平线还是想当安卓, 地平线：智驾“国产平替芯”，能扳倒英伟达吗？)
  • 公司不满足于低中阶市场，正在通过 J6P、HSD 等组合进入高算力和城市 NOA 竞争。(地平线还是想当安卓, 量产太晚是硬伤：地平线HSD能否跨越Momenta这座大山？)
  • 余凯长期公开研究英特尔、Arm、微软、安卓的发展逻辑，并明确把“生态”作为核心目标。(地平线还是想当安卓, 对话余凯：世界不止刀光剑影，是一部人来人往的江湖故事)
  • 但现有材料也反复指出，它在高阶系统量产和 HSD 上车节奏上明显晚于 Momenta 与华为乾崑。(量产太晚是硬伤：地平线HSD能否跨越Momenta这座大山？)

底层约束

  • 中国车企价格战约束: 如果只能提供性价比芯片，而不能占住更高层接口，车企会持续引入替代者压价。高置信。(地平线还是想当安卓)
  • 自研趋势约束: 车企自研芯片和算法会持续挤压独立方案商的生态空间。高置信。(地平线：智驾“国产平替芯”，能扳倒英伟达吗？)
  • 跨层升级约束: 从芯片公司升级为系统公司，不只是多做几层软件，而是要扛住客户预期、量产节奏和组织能力的重新定价。高置信。(量产太晚是硬伤：地平线HSD能否跨越Momenta这座大山？, 地平线还是想当安卓)

模型区

主模型 | 高置信

原型: 试图从国产替代穿越到默认接口权的生态型基础设施公司。

这个模型认为，地平线真正想赢的不是单颗芯片出货，而是让自己成为传统车企进入智能驾驶时代时绕不开的默认接口。 所以它会不停从芯片讲到工具链、从算法讲到生态、从量产讲到中立基础设施。余凯的个人世界观与公司的组织意图几乎是同构的。(地平线还是想当安卓, 对话余凯：世界不止刀光剑影，是一部人来人往的江湖故事, 地平线余凯：技术从来没有成为过护城河)

替代模型 | 中高置信

原型: 很强的国产替代公司，但生态叙事暂时跑在现实前面。

另一种解释是，地平线目前最确定成立的价值仍然是“基础辅助驾驶时代最成功的本土替代者”，而不是“已经形成默认接口权的生态平台”。 如果这个模型更对，那么地平线值得尊重，但其更高估值需要等待 HSD 和高算力市场真正兑现。(地平线还是想当安卓, 量产太晚是硬伤：地平线HSD能否跨越Momenta这座大山？)

创生公式 / 认知差

当前最像地平线的创生公式是:

本土芯片切入 -> 工具链与算法延伸 -> 降低传统车企智能化门槛 -> 更多客户接入 -> 生态位抬高

市场容易把地平线看成“便宜芯片供应商”或“中国英伟达”中的一种，但它真正的认知差在于: 它更像“车企智能化转型中的中立接口争夺者”。问题在于，这个接口位目前还没有被完全坐实。(地平线还是想当安卓, 地平线：智驾“国产平替芯”，能扳倒英伟达吗？)

反证 / 竞争性解释

  • 如果 HSD 等系统层能力迟迟不能形成规模量产，生态叙事会持续被削弱。(量产太晚是硬伤：地平线HSD能否跨越Momenta这座大山？)
  • 车企自研和华为模式都可能让“中立生态”变成一个更难兑现的位置。(地平线：智驾“国产平替芯”，能扳倒英伟达吗？, 量产太晚是硬伤：地平线HSD能否跨越Momenta这座大山？)

更新信号

  • 上调主模型: 如果后续材料持续证明地平线能用软硬一体方案在更多主流车企中稳定量产，主模型增强。
  • 下调主模型: 如果其高阶系统层仍长期停留在少数样板项目，而芯片价值继续主要来自低价替代，主模型要下调。
  • 上调替代模型: 如果车企自研继续加速、客户更倾向多供应商制衡、地平线生态位难以上移，替代模型会更强。
