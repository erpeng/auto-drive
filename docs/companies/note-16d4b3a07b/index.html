禾赛

禾赛最像一台把激光雷达从“贵而脆的实验室器件”压成工业部件的制造机器。它的鲜活之处不在情怀，而在一种很中国式、但又很工程化的信念: 真正的降本不是嘴上说规模，而是把设计、芯片化和自动化生产一起做穿。(对话禾赛李一帆：你仔细想行业的机会来自哪？是国家的、民族的机会)

事实底稿

  • 李一帆把激光雷达解释成“主动的眼睛”，强调它在测距与弱光环境里的确定性价值。(对话禾赛李一帆：你仔细想行业的机会来自哪？是国家的、民族的机会)
  • 公司发展的关键动作是自动化生产、ASIC 化和自研核心零部件，而不只是扩大产量。(对话禾赛李一帆：你仔细想行业的机会来自哪？是国家的、民族的机会)
  • 美国激光雷达玩家的破产与出清，反过来强化了中国制造与供应链优势的叙事。(美国败走激光雷达，最后的希望也破产了)

底层约束

  • 成本曲线约束: 激光雷达若不能持续降本，就很难守住在量产车和 Robotaxi 中的位置。高置信。(对话禾赛李一帆：你仔细想行业的机会来自哪？是国家的、民族的机会)
  • 路线之争约束: 一旦纯视觉路线在更多无人化场景里被证实足够好，独立激光雷达公司的议价会被压缩。中置信。(和楼天城聊Robotaxi：“L2越厉害，就离L4越远”, 美国败走激光雷达，最后的希望也破产了)
  • 地缘与产业约束: 禾赛的优势高度受中国制造、供应链和整车产业节奏放大。中高置信。(对话禾赛李一帆：你仔细想行业的机会来自哪？是国家的、民族的机会, 美国败走激光雷达，最后的希望也破产了)

模型区

主模型 | 高置信

原型: 激光雷达成本曲线和制造纪律的领先执行者。

这个模型认为，禾赛最重要的能力不是“证明激光雷达有用”，而是把“有用但太贵”的东西做成产业可承受的东西。 它靠的不是营销叙事，而是设计、芯片化、制造和质量体系一起推进。(对话禾赛李一帆：你仔细想行业的机会来自哪？是国家的、民族的机会)

替代模型 | 中置信

原型: 仍然受制于主路线选择的高质量部件公司。

另一种解释是，禾赛再强，也依旧受下游技术路线选择制约。 如果整车和 Robotaxi 主流路线持续压低对激光雷达的依赖，它就更像一家优秀但被架构选择约束的组件供应商。(美国败走激光雷达，最后的希望也破产了)

创生公式 / 认知差

当前最像禾赛的创生公式是:

传感器价值被验证 -> 设计与 ASIC 化 -> 自动化量产 -> 成本下降 -> 更多车型采用 -> 反哺下一代设计

市场容易把激光雷达理解成“会不会被纯视觉淘汰”的单一问题，但真正的认知差是: 只要无人化和高确定性仍重要，谁能把这种确定性做便宜，谁就有持久价值。(对话禾赛李一帆：你仔细想行业的机会来自哪？是国家的、民族的机会)

反证 / 竞争性解释

  • 如果未来主流无人化公司证明不用激光雷达也能稳定扩城，主模型会被系统性挑战。
  • 但楼天城和韩旭都在不同程度上强调激光雷达的价值，说明“很快消失”这一路径至少在当前材料里证据不足。(和楼天城聊Robotaxi：“L2越厉害，就离L4越远”, 对话文远知行韩旭：中国真正的L4只有3家，马斯克不上激光雷达干不过Waymo)

更新信号

  • 上调主模型: 如果后续材料持续证明禾赛通过芯片化和自动化获得更强成本和量产优势，主模型增强。
  • 下调主模型: 如果更多头部玩家在关键无人化场景中持续去激光雷达化，主模型要下调。
  • 上调替代模型: 如果未来更多材料显示禾赛议价能力主要受整车架构选择而非自身制造能力决定，替代模型增强。
