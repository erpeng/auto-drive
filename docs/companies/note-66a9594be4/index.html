小马智行

小马智行最鲜活的地方，是它不像一间“顺着市场热点调整姿势”的公司，而像一间死守单一目标函数的 L4 专门学校。它宁愿显得慢，也不愿把问题改写成更容易被市场理解的样子。楼天城和彭军几乎把整家公司都写成了一种方法论姿态: 不被 L2 稀释，不被端到端热词诱惑，不把 demo 当进展。(对话小马智行楼天城：只靠端到端无法通向 L4，模仿优秀司机令人绝望, 贾可对话小马智行彭军：Robotaxi九年，曾经想过的失败)

事实底稿

  • 长期坚持 L4 / Robotaxi 主线，并持续输出“L2 与 L4 不是一条路”的公开判断。(和楼天城聊Robotaxi：“L2越厉害，就离L4越远”, 对话楼天城：无人驾驶早已实现，但“牌桌”上仅三个玩家，特斯拉不算)
  • 技术路线从 Learning by Watching 转向 Learning by Practicing，明确强调世界模型、虚拟环境与强化学习。(对话小马智行楼天城：只靠端到端无法通向 L4，模仿优秀司机令人绝望)
  • 楼天城明确把 Waymo 当对手而非老师，甚至说“有一天想在客场打败 Waymo”。(对话楼天城：无人驾驶早已实现，但“牌桌”上仅三个玩家，特斯拉不算)
  • 同时也在做量产与合资等标准化动作，说明它不是完全排斥工程化，只是拒绝让量产逻辑主导目标函数。(和楼天城聊Robotaxi：“L2越厉害，就离L4越远”)
  • 王皓俊补上了更强的商业化视角: 2025 年 Robotaxi 收入 1.16 亿元，同比增长 128.6%；广州和深圳已实现城市范围单车 UE 转正；截至 2026 年 3 月 25 日车队从 2024 年底的 270 辆扩到 1446 辆，并提出年底 3000 辆、20+ 城市扩张目标。(对话小马智行CFO王皓俊：自动驾驶行业最大的竞争对手是自己)
  • 王皓俊反复强调“Top line 最重要”“不打低价策略”“先把样板打出来再扩张”，这说明小马开始更明确地用财务和运营语言解释 Robotaxi。(对话小马智行CFO王皓俊：自动驾驶行业最大的竞争对手是自己)
  • 投资人回顾材料再补了几条外部视角验证: 小马具备较高人才密度，已实现 50 万小时全无人驾驶运营和零事故；第七代 Robotaxi 硬件成本较上代下降 70%；但资本市场对其估值仍在用“单车利润 × 规模 × 区域”这类硬账来审视。(投资人回顾小马、文远口水仗：连卖车的都不会搞得这么难看)

底层约束

  • 目标函数纯度约束: 一旦被 L2 或短期量产诉求拉偏，公司最核心的自我定义就会被削弱。高置信。(和楼天城聊Robotaxi：“L2越厉害，就离L4越远”)
  • 商业时间约束: L4 商业化周期长，外界又难感知真实技术进展，这决定了它会长期承受叙事和耐心压力。高置信。(市场不信自动驾驶了，但他们还信, 对话小马智行楼天城：只靠端到端无法通向 L4，模仿优秀司机令人绝望)
  • 国际化约束: 如果真想“在客场打败 Waymo”，公司最终必须证明自己不只是中国场景玩家。中置信。(对话楼天城：无人驾驶早已实现，但“牌桌”上仅三个玩家，特斯拉不算)
  • 营收放大约束: 新材料显示，小马现在不把“单点盈利”看成最重要目标，而是把 UE 转正视为放大 Top line 的前提。这意味着它接下来要证明的不只是技术成立，而是扩张后营收增长能持续快于支出增长。高置信。(对话小马智行CFO王皓俊：自动驾驶行业最大的竞争对手是自己)
  • 规模化经济约束: 外部投资人已经开始用车均利润、空驶率、城区密度和海外单量去看小马，这说明公司正从“技术可行性考试”进入“重资产效率考试”。高置信。(投资人回顾小马、文远口水仗：连卖车的都不会搞得这么难看)

模型区

主模型 | 高置信

原型: 目标函数极纯的 L4 专营公司。

这个模型认为，小马智行最大的资产不是单次技术领先，而是长期不被错误问题诱导。它通过极强的问题边界保护了资源分配、组织认知和技术路线，因此在 L4 这件事上显得比大多数公司都更像“专门学校”。(对话小马智行楼天城：只靠端到端无法通向 L4，模仿优秀司机令人绝望, 和楼天城聊Robotaxi：“L2越厉害，就离L4越远”)

王皓俊的新访谈没有削弱这个模型，反而补了一层现实感: 小马现在开始尝试把“纯目标函数”翻译成投资人和市场能理解的语言，例如 UE、Top line、扩车和城市复制。这说明公司不是放弃主线，而是在给主线寻找经营语法。(对话小马智行CFO王皓俊：自动驾驶行业最大的竞争对手是自己) 投资人回顾材料则进一步说明，这套经营语法已经开始接受外部验证: 硬件成本下降、无人运营时长和头部城市扩张都是真的，但市场现在要求它把这些进展转成更可见的收入质量和规模效率。(投资人回顾小马、文远口水仗：连卖车的都不会搞得这么难看)

替代模型 | 中置信

原型: 技术叙事极自洽，但商业兑现周期过长的长期主义样本。

另一种解释是，小马智行的真正难点不在技术逻辑，而在这套逻辑是否能在资本、客户和公众耐心耗尽前兑现成足够大的商业结果。 如果这个模型更对，那么公司的强项仍成立，但会长期吃“太晚被市场看懂”的亏。(市场不信自动驾驶了，但他们还信, 贾可对话小马智行彭军：Robotaxi九年，曾经想过的失败)

王皓俊的表达某种程度上也在支持这个替代模型，因为他讲得更像一位正在向市场解释“为什么现在先看营收增长，而不是马上看经营性盈利”的 CFO。(对话小马智行CFO王皓俊：自动驾驶行业最大的竞争对手是自己)

创生公式 / 认知差

当前最像小马智行的创生公式是:

高保真训练环境 -> 更强无人系统 -> 更真实运营 -> 反哺训练工厂 -> 进一步拉大 L4 上限

市场更容易看到“几年过去怎么还没全面铺开”，但真正的认知差在于: 这家公司是在赌“训练工厂的正确性”，不是赌“功能演示的速度”。现在新材料又补了一层经营视角: 小马试图把这套技术飞轮进一步翻译成 UE 转正 -> 车队扩张 -> 经常性收入提升 -> 营收增长快于支出增长 的商业叙事。(对话小马智行楼天城：只靠端到端无法通向 L4，模仿优秀司机令人绝望, 对话小马智行CFO王皓俊：自动驾驶行业最大的竞争对手是自己)

反证 / 竞争性解释

  • 公司也在做量产与合资，不应被浪漫化成纯粹技术乌托邦。(和楼天城聊Robotaxi：“L2越厉害，就离L4越远”)
  • 如果未来行业事实证明 L2 数据和端到端路线足以大幅缩小差距，小马的强边界反而可能变成路径依赖。
  • 王皓俊强调“不打低价策略”“最大对手是自己”，说明公司当前还没有把竞争视角完全放到外部格局上；这既可能说明自信，也可能说明真正的大规模竞争尚未到来。(对话小马智行CFO王皓俊：自动驾驶行业最大的竞争对手是自己)

更新信号

  • 上调主模型: 如果后续材料显示其无人运营规模持续扩大，且世界模型路线的效果不断被运营结果验证，主模型更强。
  • 下调主模型: 如果公司开始大幅调整叙事，把重心迁回更容易量产的路线，或 UE 转正无法稳定复制到更多城市，主模型要下调。
  • 上调替代模型: 如果商业化和国际化持续慢于市场窗口，或者车队扩张与营收增长没有形成王皓俊所说的正循环，替代模型会增强。
