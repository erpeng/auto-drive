彭军

彭军给人的感觉，不像那种靠个人 charisma 往前冲的 CEO，更像一位对复杂系统有强耐受度的总工程师型创业者。他讲话时最鲜明的特点，是不断把外界热词拆回到问题本身，然后提醒你: 真正难的从来不是某个单点技术，而是把技术、车辆、政府、用户、运营和资本市场一起接成一条能活下去的链。(贾可对话小马智行彭军：Robotaxi九年，曾经想过的失败, 市场不信自动驾驶了，但他们还信)

事实底稿

  • 早年在百度负责自动驾驶整体战略和技术开发，之后与楼天城共同创立小马智行，属于典型的大厂高位转创业。(市场不信自动驾驶了，但他们还信, 贾可对话小马智行彭军：Robotaxi九年，曾经想过的失败)
  • 对 L2 与 L4 的界线极为执拗，明确说“通过做量产的辅助驾驶逐渐实现 L4 自动驾驶几乎完全不可能”。(市场不信自动驾驶了，但他们还信, 贾可对话小马智行彭军：Robotaxi九年，曾经想过的失败)
  • 喜欢用考试、体育、题库、100 米和 1000 米等比喻解释自动驾驶路线差异，说明他倾向把复杂抽象问题翻成可比较的训练问题。(贾可对话小马智行彭军：Robotaxi九年，曾经想过的失败)
  • 多次强调“方法是最重要的，题不重要”“数据多，所以智能高，这是最不合理的事情”，这和楼天城的世界模型取向是高度共振的。(贾可对话小马智行彭军：Robotaxi九年，曾经想过的失败)
  • 在公司经营上又非常务实，明确谈现金、上市、投资人理解、量产准备、车厂合作、合资经营与运营能力，说明他不是纯技术理想主义者。(市场不信自动驾驶了，但他们还信, 贾可对话小马智行彭军：Robotaxi九年，曾经想过的失败)

底层约束

  • 复杂度约束: 彭军反复把“复杂度极高”当作行业最大难点，说明他对系统性短板极敏感。高置信。(贾可对话小马智行彭军：Robotaxi九年，曾经想过的失败)
  • 长周期约束: 他明显接受自动驾驶是一条需要多年熬过去的路，因此不会轻易为了短期资本情绪改题。高置信。(市场不信自动驾驶了，但他们还信, 贾可对话小马智行彭军：Robotaxi九年，曾经想过的失败)
  • 组织与资源约束: 他很清楚公司能活多久、钱能烧多久、哪些分叉不能做，说明其创业观中“资源配置正确”几乎和“技术路线正确”同等重要。中高置信。(市场不信自动驾驶了，但他们还信)

模型区

主模型 | 高置信

原型: 用工程和经营双重纪律穿越高复杂度赛道的 CEO。

这个模型认为，彭军的核心能力不只是看懂技术，而是在极高复杂度下维持组织主线不散。他不像楼天城那样更像“目标函数裁判”，他更像那个要把目标函数翻译成公司行动的人。 所以他一边坚持 L4 与 L2 不是同一路，一边又不停讲资金、合作、量产、运营、政府牌照和用户体验。这类 CEO 的强项是把“不可能三角”硬拖成一个还能往前挪的系统。(市场不信自动驾驶了，但他们还信, 贾可对话小马智行彭军：Robotaxi九年，曾经想过的失败)

替代模型 | 中置信

原型: 技术信仰很强、但对技术路径有过度自信的长期主义 CEO。

另一种解释是，彭军的问题在于过于相信正确的方法论最终会赢，因而可能低估市场窗口、用户教育成本和竞争者的路径收敛速度。 如果这个模型更对，那么他的长处和盲点是同一枚硬币的两面: 他能撑住别人撑不住的长期战，也可能因此错过更现实的资源配置窗口。(贾可对话小马智行彭军：Robotaxi九年，曾经想过的失败, 市场不信自动驾驶了，但他们还信)

反证 / 竞争性解释

  • 他并非完全技术洁癖，在小马历史上也做过 Robotruck、L2++、合作造车、合资经营等现实动作，说明他会调整商业路径，只是不愿改变最终目标。(市场不信自动驾驶了，但他们还信)
  • 他对“方法不重要”的说法，本质上更像反包装而不是反方法，因此不能简单理解为他轻视新技术范式。(贾可对话小马智行彭军：Robotaxi九年，曾经想过的失败)

更新信号

  • 上调主模型: 如果后续材料持续显示彭军能在资本、政策、量产和技术之间维持主线一致，主模型更稳。
  • 下调主模型: 如果公司在关键经营节点上频繁战略摆动，或对外叙事与内部资源配置明显脱节，主模型要下调。
  • 上调替代模型: 如果未来竞争证明部分他长期否定的路径也能显著逼近 L4，替代模型会增强。
